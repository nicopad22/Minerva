# Minerva
Aplicación web diseñada por y para el Club de Cohetería Ursa Crux, basado en la Pontificia Universidad Católica.

## Objetivos principales
Minerva es una plataforma estilo ERP que tiene le fin de gestionar desde un mismo lugar todas las tareas y eventos del equipo, agilizando el desarrollo de los proyectos del club desde la organización.

Los objetivos básicos de la aplicación son:
- [x] Gestión de usuarios
- [x] Gestión de equipos
- [x] Gestión de tareas y proyectos (estructura de árbol)
- [x] Gestión de eventos
- [ ] Gestión de anuncios

A futuro, se plantea también como objetivo implementar los siguientes módulos:
- [ ] Gestión de finanzas


## Requisitos
- Node.js 22.x
- npm 10.x

## Instalación
1. Clonar el repositorio
2. Ejecutar `npm install`
3. Ejecutar `npm run dev`
